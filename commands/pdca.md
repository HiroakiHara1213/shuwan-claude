# 日次PDCA実行

SHUWANプロジェクトの日次PDCAを実行します。

`~/.claude/agents/daily-pdca.md` のワークフローに従い、全5エージェントを起動して以下を実行してください：

1. **インプット読み込み** — マスターインプット・販売実績・KPI目標を読み込む
2. **並列レビュー** — Researcher / Communication Planner / CSO / CTO が各担当領域をレビュー
3. **CMO統括** — 4エージェントの結果を統合し、KPI進捗・整合性・リスクを判断
4. **修正** — 必要に応じて各ファイルを日付付きで修正
5. **アウトプット** — 日次サマリー・変更ログを生成し、ユーザーに報告

新しいExcelデータがある場合は先に読み込んで実績を更新すること。
