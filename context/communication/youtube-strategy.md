## 戦略の意図・背景

### 解決すべき課題
- 月間購買ページ流入：1,800（目標3,000、達成率60%）
- CVR：3.5%（目標5%）
- 自社EC売上：月次目標の47%未達
- 「**知らない**」「**信じられない**」の2つの壁が購買を阻害

### なぜ「動画」なのか（YouTube単独ではなく多チャネル展開）
動画コンテンツは**1本制作すれば複数チャネルで展開できる**点が最大の強み。

動画コンテンツを以下の**3領域**に展開する：

**① 広告**（有料・新規リーチ）
- Meta動画広告（リール/フィード配信）
- PMAXの動画アセット（Google検索・ディスプレイ連動）
- YouTube広告（インストリーム配信）

**② SNS・PRコラボ**（コスト低・有機拡散）
- Instagram Reels（SNS有機拡散）
- YouTubeコラボ動画（酒泉洞堀一等の既存パートナー活用）
- 酒蔵・料飲店への投稿依頼（30蔵以上）

**③ 自社EC CVR向上**（即効・コストゼロ）
- EC商品ページへの動画埋め込み
- → CVRリフトの最速施策

- SHUWANの差別化軸（科学的根拠×プロ採用）は動画コンテンツと最高の相性
- 酒泉洞堀一チャンネル（登録者1.63万人）での**既存コラボ実績あり**（2025年実施済）

---

## 全体戦略：2層構造

```
【発見層】コンテンツ③④（コラボ動画）
→ 認知獲得・流入増
→ 月間購買ページ流入：1,800→3,000
↓
【納得層】コンテンツ①②（EC掲載・動画広告）
→ 信頼・根拠提示でCVR向上
→ CVR：3.5%→5%
↓
購買数：110碗→300碗（目標）
```

**連鎖効果**: 流入増 ← 検索増 ← クチコミ ← 成功体験 ← 良い体験動画
