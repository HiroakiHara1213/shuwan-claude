## 【最高優先】ホームページ修正タスク

### Issue 1: 「詳しく見る」ボタン リンク切れ（46回デッドクリック）
**症状**: ユーザーが「詳しく見る」ボタンをクリックしているが反応しない（デッドクリック）  
**Clarityデータ**: ClickedText="詳しく見る" / TotalDeadClicks=46  
**原因**: ボタンのhref未設定、またはクリックイベント未設定  
**修正**: 「詳しく見る」ボタンに正しいリンク先（`/shuwan/` or `/shuwan/shuwan-glass/`）を設定  
**担当**: 深田氏 / **期限: 本日中**  

### Issue 2: テキストエリアへのクリック（テキストがリンクに見えている）
**症状**: 本文テキストを多数クリックしているがリンクではない  
**Clarityデータ**:
- 「注ぐだけで味わいが開く、驚きの体験。日本酒の真価」: 99回
- 「特別な形状が、日本酒のフルーティな香りをとどめ」: 60回
- 「時代を牽引する酒蔵がテイスティングに採用。」: 43回
- 「SHUWANラインナップ」: 38回
- 「抹茶碗のもつ芸術性に加え、現代日本酒の...」: 36回

**原因分析**: テキストが視覚的にリンクのように見えている可能性（下線・色・ホバー効果）  
または、テキストをクリックして「詳しく見る」の代わりに次画面に進もうとしている  
**修正案**:
1. テキスト下の「詳しく見る」ボタンを目立つCTAに変更（色・サイズアップ）
2. 該当テキストエリア全体にリンクを付与（div全体をa要素でラップ）
**担当**: 深田氏 / **期限: 4/18**

## 【最高優先】しゅわんグラスLP修正タスク

### Issue 3: スクロール深度 30.33%（ユーザーの70%がLP下部を見ていない）
**症状**: ページを開いてもすぐ離脱。Makuakeレビュー・テレビ紹介・価格まで到達しない  
**Clarityデータ**: AvgScrollDepthPercent=30.33  
**影響**: 4/7施策で追加したMakuakeレビュー・食洗機OK表記・CTA（CVR46.9%改善の要因）が  
70%のユーザーに届いていない  
**修正案**:
1. **ファーストビューにレビュー引用を1行追加**（「Makuake3,060人支援 / ★4.8」バッジ）
2. **価格+CTAをファーストビュー内に配置**（スクロールしなくても購入できる状態に）
3. **アンカーリンクを目次風に**: 「一茂×かまいたちで紹介」「dancyu掲載」をファーストビューで見せる
**担当**: 深田氏 / **期限: 4/18**

### Issue 4: タイトル・ヘッダーへのデッドクリック（169回）
**症状**: 「日本酒の香りをひらくガラスの酒器 しゅわんグラス」のタイトルを169回クリック  
**原因**: ページタイトル（h1）をクリックして商品詳細や別ページに進もうとしている  
**修正案**: タイトル下にすぐ購入ボタンを配置（「→ Amazonで購入」「→ 公式で購入」）  
**担当**: 深田氏 / **期限: 4/18**

### Issue 5: メディア掲載情報へのクリック（40-53回）
**症状**: 「銀座しのはら」「一茂×かまいたち」「dancyu」テキストをクリックするが反応しない  
**原因**: メディア掲載情報がテキストのみで、記事リンクが貼られていない  
**修正案**: 各メディア掲載情報に元記事への外部リンクを付与（新規タブで開く）  
**担当**: 深田氏 / **期限: 4/25**
