# Context フォルダ

AIエージェントチームが参照するインプットデータの格納場所です。
自分のメモ、一次情報、考え、収集データなどをここに蓄積していきます。

## フォルダ構成

| フォルダ | 内容 | 主管エージェント |
|---|---|---|
| `market/` | 市場トレンド・競合分析・口コミ | リサーチャー / CMO |
| `strategy/` | ブランド戦略・KPI・年間計画 | CMO |
| `communication/` | 統合プラン・CRM・広告/広報/SNS | コミュニケーションプランナー |
| `sales/` | 顧客インサイト・営業施策 | CSO |
| `tech/` | プロンプト集・Claude Code活用法 | CTO |
| `memos/` | 自分のメモ・一次情報（全員参照） | 全エージェント |

## 使い方
1. 各フォルダ内の`.md`ファイルに情報を追記していく
2. エージェントに作業を依頼すると、該当フォルダのデータを読み込んで回答する
3. エージェントが調査した結果も該当フォルダに蓄積される
